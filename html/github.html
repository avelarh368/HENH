GitHub es una plataforma de alojamiento de código fuente y desarrollo de software colaborativo que utiliza el sistema de control de versiones Git. Permite a los desarrolladores gestionar, colaborar y compartir proyectos de software de manera eficiente. GitHub facilita el trabajo en equipo al permitir que varios desarrolladores trabajen en un mismo proyecto, realicen cambios, y los fusionen mediante un sistema de ramas y versiones controladas. Además, GitHub ofrece herramientas adicionales como el seguimiento de problemas, gestión de proyectos y documentación. ### Fundadores de GitHub GitHub fue fundado en 2008 por cuatro personas: 1. **Tom Preston-Werner**: Fue el creador original de la idea de GitHub y también es conocido por desarrollar el sistema de preprocesamiento de páginas estáticas Jekyll. 2. **Chris Wanstrath**: Co-fundador y ex CEO de GitHub. 3. **PJ Hyett**: Co-fundador y anteriormente CTO de GitHub. 4. **Scott Chacon**: Co-fundador y desarrollador clave en el proyecto. En 2018, GitHub fue adquirida por Microsoft por 7,500 millones de dólares en acciones.
